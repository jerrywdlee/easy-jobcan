# phantomjs 及び casperjs のインストール
## nodejsのでのインストール
```sh
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.4/install.sh | bash
$ source ~/.nvm/nvm.sh
$ nvm install node
$ npm install -g phantomjs casperjs forever
```
